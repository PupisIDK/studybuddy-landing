StudyBuddy Landing Page

Интерактивный адаптивный лендинг для приложения StudyBuddy.

О проекте

StudyBuddy — приложение для организации учебного времени студентов.
Проект разработан в рамках дисциплины:
«Тенденции проектирования графических пользовательских интерфейсов».

Функциональность

* адаптивная вёрстка
* burger menu
* плавная прокрутка
* модальные окна
* валидация формы
* hover-анимации
* scroll-анимации
* responsive design

Технологии

* HTML5
* CSS3
* JavaScript
